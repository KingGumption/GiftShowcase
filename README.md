# GiftShowcase

Static TikTok gift showcase overlays for OBS.

- Configure rewards: `Widget/public/index-rewards-config.html`
- Carousel overlay: `Widget/public/index-rewards.html`
- Rows overlay: `Widget/public/index-rewards-rows.html`

The widget is designed to work as static files, so it can be hosted on GitHub Pages or another static host and pasted into OBS as a browser-source URL.
